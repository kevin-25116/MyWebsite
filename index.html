What is digital media?

According to Wikipedia digital media are any media that can be created, viewed, changed, stored and shared on digital devices. Digital media is typically accessed using an App or website.

So what is media?

Media includes a lot of things but are usually images, text, audio, video and databases. When combined together this means media includes animations, movies, web series, TV, radio, games, books, podcasts, news articles, websites, instruction manuals and more. Media is basically everything we create, change, share, view and store. Digital media is everything we create, change, share, view and store using an App or website.
